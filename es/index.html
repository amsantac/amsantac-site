Mapeo web con Leaflet y R

Crea mapas web interactivos con unas cuantas líneas de R

Leaflet es una librería de JavaScript my popular para la creación de mapas interactivos. Una forma de crear un mapa usando la librería JavaScript de Leaflet es incluir los archivos de la librería Leaflet y los archivos de estilo (CSS) en la cabecera de la página y luego definir la configuración del mapa en el cuerpo de la página html.

Una forma alternativa es crear el mapa web usando un paquete desarrollado para el lenguaje R llamado leaflet, desarrollado por el grupo de RStudio, el cual permite controlar e integrar mapas de Leaflet en R. En esta entrada del blog explico cómo leer en R un mapa vectorial en formato shapefile y cómo crear un mapa web con leaflet personalizando la forma en que el mapa vectorial es presentado. También describo cómo adicionar una legenda, un control para las capas y popups para desplegar datos de la tabla de atributos.

Aplicación web TOC

Una aplicación interactiva para generar las curvas ROC y TOC desarrollada con Shiny

Esta aplicación web desarrollada con Shiny y R permite la construcción de la curva ROC (i.e., Receiver (aka Relative) Operating Characteristic) y la curva TOC (i.e., Total Operating Characteristic) para datos espaciales y no espaciales a través de una interfaz gráfica de usuario. La curva ROC es una técnica estadística muy popular usada para evaluar el comportamiento de un sistema clasificador binario a medida que el umbral de discriminación varía. El método TOC es una modificación novedosa de la técnica ROC que ha sido desarrollado recientemente por el profesor Gil Pontius de Clark University.

Redes óptimas de muestreo espacial

Diseño de redes de muestreo usando algoritmos genéticos

Este capítulo de libro presenta cómo se diseñaron redes espaciales óptimas para la segunda fase de muestreo del carbono orgánico del suelo con base en la información espacial obtenida durante una primera fase de muestreo en campo. Puntos adicionales complementarios al muestreo inicial existente fueron generados en configuraciones regulares y aleatorias siguiendo dos métodos diferentes: adición secuencial y adición simultánea. La búsqueda del conjunto óptimo de puntos adicionales se llevó a cabo usando una técnica de optimización evolutiva conocida como algoritmos genéticos.
